📝 Todo App (React)
This is a simple and responsive Todo application built using React. The project allows users to efficiently manage their daily tasks by adding, editing, marking as completed, and deleting todos.
The main goal of this project was to practice core React concepts such as state management, component structure, and event handling while building a real-world usable application.
🚀 Features
Add new tasks
Mark tasks as completed / active
Delete tasks
Simple and clean UI
Responsive design for different devices
🛠️ Tech Stack
React (Functional Components)
JavaScript (ES6+)
HTML & CSS
